Análisis y Proyección hacia IoT

Propuesta técnica para transformar el sistema de riego hidropónico de Riohacha en un ecosistema inteligente interconectado:

¿Qué datos genera el dispositivo?
Genera métricas de temperatura, humedad ambiental, humedad del sustrato y estado de bomba. Escalable a pH y conductividad.
¿Cómo enviaría datos a internet?
Vía ESP32 (WiFi) usando protocolos MQTT o HTTP REST hacia nubes IoT como Firebase o Brokers especializados.
¿Cómo visualizarían esos datos?
Mediante este Dashboard Web de alta fidelidad, permitiendo supervisión remota total desde cualquier dispositivo.
¿Cómo activarían algo remoto?
Con botones en el dashboard que envían comandos de vuelta al ESP32 para accionar el Relay en GPIO 13.
Flujo Completo
Sensor → ESP32 → WiFi → Broker Cloud → Dashboard → Comando Usuario → Actuador.
Propuesta de Valor
Trasladar la inteligencia local a la nube para gestión agrícola sin fronteras geográficas.
